# 🎮 Multiplayer RPG

## Objetivo

Projeto desenvolvido para aplicação dos conhecimentos de:

- Backend com Python
- FastAPI
- Comunicação em tempo real
- Arquitetura de sistemas multiplayer
- Gerenciamento de projetos

## Tecnologias

### Backend
- Python
- FastAPI
- WebSocket

### Frontend / Game Client
- Godot 4
- GDScript
